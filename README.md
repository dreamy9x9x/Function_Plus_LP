# 🚀 Function Plus

Landing page moderna para apresentação de serviços e projetos.



# 🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript



---

# 📦 Como Usar

Opção 1: GitHub Pages (Recomendado)

1. Faça um fork ou clone do repositório


2. Vá em Settings > Pages


3. Em Source, selecione:

Branch: main

Pasta: / (root)



4. Aguarde alguns minutos


5. Acesse o site gerado




---

Opção 2: Local

1. Clone o repositório:



git clone https://github.com/seu-usuario/function-plus.git
cd function-plus

2. Abra o arquivo:



index.html


---

# 🎨 Personalização

Alterar cores

Edite as variáveis no início do arquivo index.html:

:root {
    --color-bg: #0A0E17;
    --color-primary: #00E5CC;
    --color-accent: #FF6B6B;
}


---

Modificar conteúdo

Edite o arquivo index.html:

<section class="hero"> → Banner principal

<section id="services"> → Serviços

<section id="contato"> → Contato



---

Formulário de contato

Para funcionar, conecte com um backend ou serviço como:

Formspree

EmailJS

Firebase

# 🐛 Problemas comuns

Site não aparece

Aguarde 1–5 minutos

Limpe o cache

Verifique se o branch está correto


Erro 404

Confirme que existe:


index.html
