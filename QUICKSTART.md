

# 🚀 Quick Start — Function Plus

Guia rápido para colocar seu site no ar com GitHub Pages.


---

# 📦 1. Criar repositório

1. Acesse o GitHub


2. Clique em New repository


3. Nome:



function-plus

4. Marque: Public


5. Clique em Create repository




---

# ⬆️ 2. Enviar arquivos

Via terminal:

# 1. Navegue até a pasta do projeto
cd caminho/para/function-plus

# 2. Inicialize o Git
git init

# 3. Adicione os arquivos
git add .

# 4. Commit inicial
git commit -m "Initial commit"

# 5. Defina a branch principal
git branch -M main

# 6. Conecte ao repositório remoto
git remote add origin https://github.com/seu-usuario/function-plus.git

# 7. Envie os arquivos
git push -u origin main


---

# 🌐 3. Ativar GitHub Pages

1. Vá em Settings


2. Clique em Pages


3. Em Source:

Branch: main

Folder: / (root)



4. Clique em Save




---

# ⏱️ 4. Acessar o site

Após alguns minutos, acesse:

https://seu-usuario.github.io/function-plus/


---

# 🔄 Atualizações futuras

git add .
git commit -m "Atualização"
git push


---

# ⚠️ Observações

Certifique-se de que existe um arquivo index.html

Aguarde até 5 minutos para publicação

Limpe o cache se o site não carregar
